# Créditos e licenças dos assets (protótipo)

Estes arquivos são de terceiros, usados apenas no protótipo (`main-3d-explorer.html`).

## Modelo 3D — `assets/jeep.glb`
- Veículo: **Jeep** (marca pertencente à Stellantis), usado como modelo de
  veículo **real** para o protótipo.
- Origem: asset 3D de terceiro obtido publicamente para fins de prototipagem.
- Licença: **A CONFIRMAR.** É um placeholder para validar a experiência 3D.
  Antes de qualquer uso em produção ou distribuição, a licença/origem do modelo
  deve ser verificada — ou substituída por um modelo oficial/licenciado.
- Troca: qualquer `.glb` pode substituí-lo; a escala e o enquadramento da câmera
  se ajustam automaticamente no código.
